# 01 Start / Room Entry

Image mapping: `../images/scene-01-start-room-entry.svg`

## Purpose

Entry screen for creating or joining a multiplayer room.

## Layout

- Top bar: menu icon on the left, settings and help icons on the right.
- Center content:
  - Crown icon
  - Title: `PROJECT GO-ROYALE`
  - Subtitle: `4-Player Tactical Battle`
  - Primary button: `CREATE ROOM`
  - Secondary button: `JOIN ROOM`
- Bottom status bar:
  - Online status
  - Version: `v0.1.0 (MVP)`

## Primary Actions

- `CREATE ROOM` opens scene 02.
- `JOIN ROOM` opens scene 03.

## State Notes

- User is online.
- No active room selected.
