# Join Lobby Wireframe

## Purpose

Minimal join-room lobby screen for a player who already has or receives a room code.

## Scope

- One text input field for entering a room code.
- One primary button labeled `Join Room`.
- No title, help text, validation copy, room list, player preview, settings, or secondary actions in this first sketch.

## Layout

Target format: mobile portrait wireframe.

```text
+--------------------------------+
|                                |
|                                |
|                                |
|                                |
|                                |
|                                |
|      +------------------+      |
|      | Room Code        |      |
|      +------------------+      |
|                                |
|      +------------------+      |
|      |    Join Room     |      |
|      +------------------+      |
|                                |
|                                |
|                                |
|                                |
|                                |
|                                |
+--------------------------------+
```

## Elements

| Element | Text | Behavior |
| --- | --- | --- |
| Text input | `Room Code` placeholder | Accepts typed/pasted room code. |
| Button | `Join Room` | Attempts to join the entered room. |
